# 📋 ロール管理アプリ

シンプルで使いやすいロール管理アプリケーションです。

## 機能

✅ ロールの作成・編集・削除
✅ ロール説明の管理
✅ 権限レベルの設定（管理者・編集者・閲覧者）
✅ ブラウザ内蔵ストレージでデータ保存
✅ レスポンシブデザイン対応
✅ モダンなUI/UX
✅ **PWA対応 - スマホのホーム画面に追加可能**
✅ **オフライン対応 - インターネット接続がなくても動作**

## 使い方

1. ロール名を入力
2. 説明を入力
3. 権限レベルを選択
4. 「追加」ボタンをクリック

### 編集・削除

- 各ロールの「編集」ボタンでロール情報を変更
- 「削除」ボタンでロールを削除

## PWAとして使用する

### スマホのホーム画面に追加

**iPhone / iPad:**
1. Safari で `https://YOUR_USERNAME.github.io/roll-app/` を開く
2. 共有ボタン → 「ホーム画面に追加」

**Android:**
1. Chrome で `https://YOUR_USERNAME.github.io/roll-app/` を開く
2. メニュー → 「アプリをインストール」または「ホーム画面に追加」

### オフライン機能

- Service Worker が有効なため、一度アクセスするとオフライン状態でも動作します
- ロールデータはブラウザに保存されるため、インターネット接続なしで使用可能

## 技術仕様

- HTML5 + CSS3 + JavaScript
- LocalStorageでブラウザ内にデータを保存
- Service Worker でキャッシング と オフライン対応
- Web App Manifest (PWA対応)
- レスポンシブデザイン対応（PC・タブレット・スマホ）

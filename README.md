# 📋 スタンド管理アプリ

スタンド番号ごとに、現在のステータスやメモを管理できるシンプルなWebアプリです。
ブラウザだけで使えるため、専用ソフトをインストールしなくても利用できます。

## 主な機能

- **検索**
  スタンド番号、ステータス、メモの内容から、登録済みデータをすばやく探せます。

- **追加**
  新しいスタンド番号、ステータス、メモを登録できます。

- **編集**
  登録済みのスタンド情報をあとから変更できます。

- **削除**
  不要になったスタンド情報を削除できます。

- **CSVエクスポート**
  登録データをCSVファイルとして書き出せます。バックアップや表計算ソフトでの確認に使えます。

- **CSVインポート**
  CSVファイルからスタンド情報をまとめて取り込めます。

- **Google Sheets同期**
  Google Apps Scriptを使って、Googleスプレッドシートとデータを同期できます。複数端末で同じデータを扱いたい場合に便利です。

- **PWA対応**
  スマートフォンやPCのホーム画面に追加して、アプリのように起動できます。一度読み込んだ画面は、オフラインでも表示できます。

## 使い方

### 1. ログインする

アプリを開くとログイン画面が表示されます。
パスワードを入力してログインすると、スタンド管理画面が表示されます。

### 2. スタンド情報を追加する

1. 「スタンド番号」を入力します。例: `#2-11`
2. 「ステータス」を選択します。
3. 必要に応じて「メモ」を入力します。
4. 「追加」ボタンを押します。

### 3. 登録済みデータを検索する

画面上部の検索欄に、スタンド番号、ステータス、またはメモの一部を入力します。
入力した内容に一致するデータだけが一覧に表示されます。

### 4. スタンド情報を編集する

一覧の「編集」ボタンを押すと、登録済みの内容を入力欄に呼び出せます。
内容を変更したあと、「更新」ボタンを押すと保存されます。

### 5. スタンド情報を削除する

一覧の「削除」ボタンを押すと、そのスタンド情報を削除できます。
削除前に確認メッセージが表示されます。

### 6. CSVでデータを出し入れする

- 「CSVエクスポート」ボタンで、現在のデータをCSVファイルとして保存できます。
- 「CSVインポート」ボタンで、CSVファイルの内容をアプリに取り込めます。

CSVは、バックアップを取りたいときや、Excel・Googleスプレッドシートなどで編集したいときに便利です。

## Google Sheets同期について

このアプリは、Google Apps Scriptを通してGoogleスプレッドシートと同期できます。

同期が成功すると、ブラウザ内のデータだけでなく、スプレッドシート側にも同じ内容が保存されます。
ネットワークエラーなどで同期に失敗した場合でも、データはブラウザ内に保存されます。


### スプレッドシートの見た目

同期時にGoogle Apps Script側で、スプレッドシートを自動で見やすく整えます。

- ヘッダー行を太字・背景色付きにします。
- 1行目を固定して、下にスクロールしても項目名を確認できるようにします。
- ステータス列に条件付き書式を設定し、「オンライン」「改削中」「中古予備」「その他」を色分けします。
- 初回同期時だけ、ID=60、スタンド番号=120、ステータス=190、メモ=260、最終更新日=180 の列幅に設定します。
- 一度設定したあとは、手動で変更した列幅を同期時に上書きしません。
- フィルターと罫線を設定します。

## PWAとして使う

PWA対応により、スマートフォンやPCのホーム画面に追加できます。
ホーム画面に追加すると、通常のアプリのように起動しやすくなります。

### iPhone / iPadの場合

1. Safariでアプリを開きます。
2. 共有ボタンを押します。
3. 「ホーム画面に追加」を選びます。

### Androidの場合

1. Chromeでアプリを開きます。
2. メニューを開きます。
3. 「アプリをインストール」または「ホーム画面に追加」を選びます。

## データの保存場所

このアプリでは、主に次の2か所にデータを保存します。

1. **ブラウザ内の保存領域（LocalStorage）**
   使っている端末のブラウザ内にデータを保存します。

2. **Googleスプレッドシート**
   Google Sheets同期が設定されている場合、スプレッドシートにもデータを保存します。

ブラウザのデータを削除すると、LocalStorageに保存されたデータも消える場合があります。
大事なデータは、CSVエクスポートやGoogle Sheets同期でバックアップすることをおすすめします。

## 技術仕様

- HTML / CSS / JavaScriptで作られた静的Webアプリ
- LocalStorageによるブラウザ内保存
- Google Apps ScriptによるGoogle Sheets同期
- CSVインポート・エクスポート対応
- Service Workerによるキャッシュ対応
- Web App ManifestによるPWA対応
- PC、タブレット、スマートフォンで使えるレスポンシブデザイン

## ファイル構成

```text
roll-app/
├── index.html              # メイン画面のHTML
├── css/
│   └── styles.css          # 画面デザイン用のCSS
├── js/
│   ├── app.js              # 画面操作・データ管理の処理
│   ├── csv.js              # CSVインポート・エクスポート処理
│   └── sync.js             # Google Sheets同期処理
├── manifest.json           # PWA設定ファイル
├── service-worker.js       # キャッシュ・オフライン表示用の処理
├── google-apps-script.gs   # Google Sheets同期・シート整形用のApps Script
├── 404.html                # ページが見つからない場合の画面
└── README.md               # この説明ファイル
```

## デプロイ

このアプリは、GitHub Pagesなどの静的サイトホスティングで公開できます。
Google Sheets同期を使う場合は、Google Apps Script側の設定も必要です。
